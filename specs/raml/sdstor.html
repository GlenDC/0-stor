Secured by itsyouonline with scopes:
  • user:name

OAuth2 is a protocol that lets external apps request authorization to private details in a user's account without getting their password.

  • Request
  • Response
  • Security

URI Parameters

  • nsid: required (string)

Body

Media type: application/json

Type: object

Properties
  • size: required (integer - minimum: 1)

    Storage size you want to reserve in MiB

  • period: required (integer - minimum: 1)

    Number of days the reservation is valid

Example:

properties:

{
  "size": 128,
  "period": 30
}

HTTP status code 200

Reservation created. The response contains two tokens. reservationToken, used to managed this reservation. dataAccessToken, used to perform action on the data in this reservation

Body

Media type: application/json

Type: object

Properties
  • reservation: required (object)
    • id: required (string - pattern: ^[a-zA-Z0-9]+$)
    • adminId: required (string - pattern: ^[a-zA-Z0-9]+$)

      ItYou.Online ID of the user that has right to administrate this reservation

    • created: required (datetime)
    • updated: required (datetime)
    • sizeReserved: required (number - minimum: 1)

      Storage size of the reservation in MiB

    • sizeUsed: required (number - minimum: 1)

      Storage size of the reservation already used

    • expireAt: required (datetime)

      Expiration date

  • reservationToken: required (string)
  • dataAccessToken: required (string)

HTTP status code 403

The reservation has not been created. Possible reason, not enough space left on the server, ... Inpsect error message for more detail

Body

Media type: application/json

Type: object

Properties
  • error: required (string)
